# wacai-receipt-import

解析含收支记录的图片（截屏/拍屏）并自动导入挖财 APP

## 使用示例

```
@skill:wacai-receipt-import 解析收支图片(当前：2026年)上传至挖财日常账本 -H 'X-Access-Token: WCeO2k48nYJsgxxxxxxxNehzk5o-7V7Q' \ 收付账户：交通信用卡 2690
```

## 说明

1. 部分 AI 解析年份不准（在截图没有明确的年份的时候，比如可能解析成 2025 年，所以需要声明当前的年份）
2. 需要选择一个账本（名称），比如：日常账本
3. 需要提供挖财网页端登录后的 X-Access-Token，登录后 F12 通过 Fetch/XHR 可以在 headers 中查看
   - 地址：`https://jz.wacaijizhang.com/jz-pc/flow`
4. 需要明确收付账户，一般建议有 4 位数卡号尾号，否则挖财接口匹配不到账户（但依然会导入，从报表、月份详情中能查到）

## 注意事项

AI 解析的优势是快速提取关键信息，但因为银行收支记录本身就模糊，比如有些看不到消费去向，所以识别出来的结果也不一定完全达到你的预期，你可以在导入后在 APP 中继续调整，总之比你一笔笔录入要省心很多。

建议一次上传 1-3 张图片（建议不要滚动截屏，或高度不超过两屏衔接高度，图片数量 5 张以内），大概 20 笔记录以内
